# v1.0.0

首个正式发布版本，聚焦饮水提醒核心体验与本地稳定性。

## 亮点
- 沉浸式首页与饮水记录体验优化。
- 完整的提醒设置流程：间隔提醒、场景提醒、免打扰时段。
- Android 本地提醒能力增强，支持在本地稳定触发提醒。

## 提醒相关更新
- 场景提醒支持新增、编辑、删除与开关，并进行持久化保存。
- 修复场景提醒“新增后闪现又消失/开关回弹”的问题。
- 修复场景名称清空后被自动回填默认文案的问题。
- 在“默认提醒间隔”模块中新增“下一次提醒时间”显示。
